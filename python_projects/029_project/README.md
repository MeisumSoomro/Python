# Inventory Management System

A desktop application for managing inventory, tracking sales, and generating reports.

## Features

- Product management (add, edit, delete)
- Stock tracking
- Low stock alerts
- Barcode scanning and generation
- Sales tracking
- Report generation
  - Stock reports
  - Sales reports
  - Low stock reports
- Data import/export
- SQLite database storage
