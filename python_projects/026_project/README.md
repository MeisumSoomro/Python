# Music Player Application

A desktop music player application with visualization, built with Python and Tkinter.

## Features

- Play MP3 files
- Create and manage playlists
- Audio visualization
- Basic playback controls
- Volume control
- Track progress bar
- ID3 tag support
- Audio visualization
- Playlist import/export
