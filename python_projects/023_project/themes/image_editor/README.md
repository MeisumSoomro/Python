# Image Editor

A feature-rich image editor built with Python using PySimpleGUI and Pillow.

## Features
- Basic image operations (open, save)
- Image filters (blur, sharpen, contour, emboss, edge enhance)
- Brightness and contrast adjustment
- Image transformation (rotate, resize)
- Undo/Redo functionality
- Original image preservation

## Requirements
- Python 3.x
- PySimpleGUI
- Pillow (PIL)
